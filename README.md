# Config Editor Application

WPF application written in C# that reads configuration data from an XML file, allows editing it through a GUI and saves the changes.

## Features

- Reads data from config.xml
- Displays accounts in a GUI
- Allows editing of account data
- Saves changes back to XML
- Generates 12-character passwords using a Bash script
- Supports selecting multiple users

## Technologies

- C#
- WPF
- LINQ to XML
- Bash script

## How to run

1. Clone the repository
2. Open EditorApp.sln in Visual Studio
3. Run the application

Note: The application requires Git Bash installed because the password generation uses a Bash script.
